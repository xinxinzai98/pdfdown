## 1. 官方网站渠道

### 1.1 出版社官网

**主要出版社及其官方下载：**

| 出版社 | 官网 | 说明 |
|--------|------|------|
| **Elsevier (ScienceDirect)** | https://www.sciencedirect.com/ | 需要机构订阅 |
| **Springer** | https://link.springer.com/ | 部分开放获取 |
| **Wiley** | https://onlinelibrary.wiley.com/ | 需要订阅 |
| **IEEE Xplore** | https://ieeexplore.ieee.org/ | 需要订阅 |
| **ACM Digital Library** | https://dl.acm.org/ | 需要订阅 |

### 9.2 arXiv 下载脚本

```python
import requests

def download_arxiv(arxiv_id, filename=None):
    """下载 arXiv 论文"""
    url = f"https://arxiv.org/pdf/{arxiv_id}.pdf"
    
    response = requests.get(url)
    
    if filename is None:
        filename = f"{arxiv_id}.pdf"
    
    with open(filename, 'wb') as f:
        f.write(response.content)
    
    print(f"Downloaded: {filename}")

# 使用
download_arxiv("2312.12345")
```

### 9.3 Semantic Scholar API 脚本

```python
import requests
import json

def get_semantic_scholar_papers(query, max_results=10):
    """查询 Semantic Scholar"""
    url = f"https://api.semanticscholar.org/graph/v1/paper/search"
    
    params = {
        'query': query,
        'limit': max_results,
        'fields': 'title,authors,openAccessPdf,year'
    }
    
    response = requests.get(url, params=params)
    data = response.json()
    
    for paper in data.get('data', []):
        print(f"\nTitle: {paper.get('title')}")
        print(f"Year: {paper.get('year')}")
        
        if paper.get('openAccessPdf'):
            pdf_url = paper['openAccessPdf']['url']
            print(f"PDF: {pdf_url}")

# 使用
get_semantic_scholar_papers("machine learning")
```
